# PDF Question Answering Chatbot

This project implements a Retrieval-Augmented Generation (RAG) chatbot that answers questions based on a PDF document.

## Features
- Reads PDF documents
- Retrieves relevant content using FAISS
- Answers questions from the document
- Returns "Not found in document" for unrelated queries

## Technologies
- Python
- LangChain
- FAISS
- Sentence Transformers
- Streamlit

## Run

```bash
streamlit run streamlit_app.py
